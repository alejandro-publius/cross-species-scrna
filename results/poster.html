Pipeline

1
Data
Baron GSE84133 raw UMI counts, author cell-type labels; QC + log-normalize.
2
Shared gene space
MGI homology → strict 1:1 orthologs. Case-aware (INS vs Ins1); INS itself excluded as a co-ortholog — a real 1:1 bias.
3
VAE from scratch
Raw PyTorch, negative-binomial ELBO — comprehension insurance.
4
scVI integration
Species as the batch covariate → shared latent that mixes species globally, keeps cell types.
5
Supervised contrastive
Trained on human labels only (mouse held out) to test cross-species transfer honestly.

Honest limitations

  • Local mixing (iLISI/kBET) is modest — integration is global, not fine-grained.
  • Strict 1:1 orthologs drop duplicated families (e.g. INS) — excluded, not "specific".
  • Balanced acc (0.90) < raw acc (0.965): rare types (schwann, macrophage) transfer worse.
  • Conserved/specific is descriptive marker overlap, not a causal claim.
